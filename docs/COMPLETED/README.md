# Completed Phases — Historical Reference

This folder contains documentation for phases that have shipped and are no longer in active development. They are kept as historical context and for reference when understanding design decisions.

## Contents

- **phase-4-plan.md** — Phase 4 (Home Network + Multi-User) implementation plan. Details the Node.js backend, SQLite database, multi-user auth, per-user data isolation, and deployment strategy. Shipped in v1.0.

- **phase-5b-sharing-plan.md** — Phase 5b (Household Sharing) implementation plan. Details household groups, shared accounts with permission control, transaction splits with settlement tracking, and all design decisions locked per owner sign-off. Shipped in v1.0.1.
